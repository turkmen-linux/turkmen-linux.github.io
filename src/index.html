What is Türkmen Linux ?

Türkmen Linux is a independent GNU/Linux distribution. It uses its own YMP (Yerli ve Milli Package) manager. Türkmen Linux is a distribution originating from Turkiye. 🇹🇷 The reason for its development is to create an independent distribution using new generation technologies and to provide ease of use.

All packages come with header files and static libraries. In this way, it is easier to compile than other distributions.
Turkman does not use systemd. It prefers openrc as init system.
It has a mixed (binary and source) package system that works quickly. While those who want to install packages from the source code, those who want can use the compiled packages.

Türkmen linux is hardened by default. (Users can disable hardening if they wants.)
